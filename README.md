# Breast-cancer-gene-exprission-python
The study focuses on **Breast Cancer**, utilizing transcriptomic data to investigate molecular differences in gene expression between cancerous and healthy breast tissue.

**Gene expression** data measures the activity level of genes, indicating how much **mRNA** is produced by a cell under specific biological conditions.


Breast cancer tissue is fundamentally different from normal tissue due to genetic dysregulation; while 

**normal breast tissue** maintains stable and balanced gene expression to ensure healthy cellular function,

**cancer tissue** exhibits uncontrolled growth, where oncogenes are overexpressed, and tumor suppressor genes are suppressed.


The study utilized **Microarray technology** (specifically the Affymetrix platform) to quantify gene expression profiles of **104** breast cancer and **17** normal breast biopsies. This comparison between cancer and normal tissue allows researchers to identify Differentially Expressed Genes (DEGs), which act as biomarkers for early detection and potential therapeutic targets.
# Exploring Gene Expression Differences Between Breast Cancer and Normal Breast Tissue
## Project Overview
This beginner bioinformatics project explores selected gene expression differences between breast cancer and normal breast tissue using Python.
## Dataset
The project is based on the public GEO dataset GSE42568.
## Aim
The aim is to practice basic biological data handling, simple comparison, visualization, and biological interpretation.
